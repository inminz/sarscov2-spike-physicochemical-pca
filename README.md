# SARS-CoV-2 Spike Physicochemical PCA

Interactive 3D PCA visualization of SARS-CoV-2 spike protein physicochemical properties across multiple countries.

This project presents a dimensionality reduction analysis based on key physicochemical features, including:
- Charge at pH 7.0  
- Hydropathy  
- Thermostability (aliphatic index)

The visualization allows:
- Exploration of cluster structure in PCA space  
- Comparison between WHO-designated variants and physicochemical clusters  
- Interactive inspection of individual sequences and their properties  

## Interactive Visualization

The interactive 3D PCA plot is available via GitHub Pages:

👉 https://inminz.github.io/sarscov2-spike-physicochemical-pca/

## Notes

- The visualization is generated using R (plotly, htmlwidgets)
- The dataset includes spike protein sequences from multiple countries
- PCA is performed on scaled physicochemical variables
